`uri-kit` is a tiny, typed library for opening native surfaces on Apple
platforms — iOS settings panes, system app schemes, universal links, and
`x-callback-url`. It ships with a catalog of 100+ first‑party URIs, a CLI,
a React entrypoint, and a bundled MCP server so AI agents can compose URIs
as well.

## Quickstart

Install `uri-kit`:

```bash
npm install uri-kit
```

Define a URI as a plain object and call it:

```ts
import { defineURI } from "uri-kit";

const openWiFi = defineURI({
  scheme: "prefs",
  path: "root=WIFI",
  platforms: ["ios", "ipados"],
});

openWiFi();
```

catalog, and a live MCP server. See the [agents overview](/agents).
