## 0.1.0 — initial release

- Declarative `defineURI` API with typed fallbacks.
- 100+ first‑party catalog entries covering iOS Settings, Apple system
  apps, macOS System Settings, and x‑callback‑url.
- CLI: `add`, `init`, `list`, `check`, `search`, `build`, `llms`, `mcp`.
- React entrypoint: `useURI`, `usePatch`, `URIKitProvider`.
- `uri-kit/agent` export for LLM‑time catalog lookup.
- Bundled MCP server (`npx uri-kit mcp`) with four tools.
- `llms.txt` + `llms-full.txt` + `api.json` generators.
- `SKILL.md` for Cursor / Claude agent auto‑discovery.
